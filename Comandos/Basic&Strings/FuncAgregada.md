# Funções Agregadas

- ```max()```
- ```min()```
- ```sum()```
- ```avg()```

**Tabela**

    | id | nome | salario |
    | -- | ---- | ------- |
    | 1  | Mark | 1500.65 |
    | 2  | Niki | 2360.96 |
    | 3  | Ghil | 1000.00 |

**Comando MAX()**
    SELECT max(salario)
    FROM funcionario as f;

**Retorno**

    | salario |
    | ------- |
    | 1500.65 |

**Comando MIN()**

    SELECT min(salario)
    FROM funcionario as f;

**Retorno**

    | salario |
    | ------- |
    | 1000.00 |

**Comando SUM()**

    SELECT min(salario)
    FROM funcionario as f;
