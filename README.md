# Banco de Dados
Nesse repositório irei falar sobre os comandos abordados em aula, cada aula possuirá um arquivo semelhante a este explicando linha por linha e comando por comando, além dos conceitos abordados em sala de aula;

> Claro, na medida do possível, tentarei explicar brevemente de coisas que o professor não abordou justamente por causa do tempo que ele tem em sala;

# Acessando as Explicações
Cada pasta: ```Aulas```, ```Exercicios``` e ```PreAulas``` terão mais uma pasta com o nome ```Explicacoes```, onde terá um ou mais arquivos ```.md``` explicando a maioria dos comandos abordados nos exercícios
